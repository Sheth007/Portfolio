Brief Description

Indie Vibes's frontend is developed using Html, Css & JavaScript,
And backend is developed using Php, JavaScript & MySql,
In short Indie Vibes is free to use web music player where listening is totaly free not like others,
User have to just register/signup and then they are ready to listen,
User can play random songs or select any artist they want to listen,
In future user's can upload songs and publish it on Indie Vibes.
