🕵 Flutter Recon

Published · Browser Extension

Overview

Flutter Reacon is an advanced developer utility designed to instantly identify whether websites are built using Google's Flutter web framework. Built for developers, reverse engineers, and tech stacks researchers, it unmasks framework packages in real time.

Key Highlights

  • Automated Detection: Monitors browser tabs to identify active instances of Flutter web configurations.
  • Dependency Diagnostics: Extracts and prints internal packages actively used by the production build.
  • Direct Integration Links: Generates deep links matching packages directly onto pub.dev for isolated package investigation.
  • Zero Setup Footprint: Functions in one click directly from your extension toolbar panel.
